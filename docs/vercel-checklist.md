# Vercel Checkliste

1. GitHub-Repository erstellen oder pruefen
2. Projekt pushen
3. Vercel oeffnen
4. `Add New Project` waehlen
5. Repository importieren
6. Framework auf `Vite` setzen
7. Build Command auf `npm run build` setzen
8. Output Directory auf `dist` setzen
9. Environment Variablen setzen:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_PUBLISHABLE_KEY`
10. Deploy starten
11. Vercel-URL oeffnen
12. In Supabase die Redirect URLs auf die Vercel-Domain erweitern
13. Login testen
14. Kernflow testen:
   - Login als Veranstalter
   - Event erstellen
   - Event veroeffentlichen
   - Event oeffentlich pruefen
   - Haendler zuordnen
   - Event wieder intern setzen
